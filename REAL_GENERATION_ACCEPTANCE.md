# REAL_GENERATION_ACCEPTANCE.md

## Retired: No User-Facing Live Generation Switch

This acceptance log is retired as of 2026-06-05.

Poster Lab Pro no longer exposes a user-facing `真实生成` switch, `确认真实生成保护` action, manual verification control, or manual live-test route. Future release planning should use `USER_TESTING.md`, `MULTIMODE_ACCEPTANCE.md`, `DESKTOP_TESTING.md`, and `TESTING.md` instead.

Provider execution remains part of the normal generation workflow when credentials and routes are configured, but there is no separate real-generation switch to enable or verify.
